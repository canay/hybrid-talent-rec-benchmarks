# Raw public source retrieval

This replication package includes the frozen prepared benchmarks used in the manuscript. The original raw public source files are intentionally retrieved from their original providers rather than mirrored here.

## Official source entry points

- JobHop dataset page: <https://huggingface.co/datasets/aida-ugent/JobHop>
- JobHop project page: <https://esco.ec.europa.eu/en/about-esco/publications/publication/jobhop-large-scale-dataset-career-trajectories>
- Karrierewege dataset page: <https://huggingface.co/datasets/ElenaSenger/Karrierewege>
- Karrierewege project repository: <https://github.com/elenasenger/karrierewege>

## Exact file URLs used for benchmark freezing

The benchmark-specific metadata files record the pinned direct URLs used during preparation:

- `../prepared/jobhop_ict_benchmark_v1/metadata.json`
- `../prepared/karrierewege_ict_benchmark_v1/metadata.json`

These metadata files also preserve the benchmark filtering rules and summary statistics needed to verify that the prepared benchmarks match the manuscript version.
